# Képernyőképek

A főoldali `README.md` ezeket a fájlokat hivatkozza. Tedd ide a PNG-ket ezeken a neveken:

| Fájl | Mit mutasson |
|---|---|
| `login.png` | Bejelentkezés képernyő (felhasználó + TOTP), branding-gel |
| `devices.png` | Eszközlista (online, felhasználó, publikus IP, verziók) + jobbklikk-menü |
| `commands.png` | A Parancsok és/vagy Üzenetek fül egy eszköznél |
| `settings.png` | Szerver beállítások (branding + e-mail) |

Tipp: ~1280px széles, sötét téma jól mutat. Ha más nevet/elrendezést akarsz, írd át a `README.md` „Képernyőképek" szakaszát.
